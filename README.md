### Description
- This is a cli tool to get instagram's followers':
    - ids 
    - usernames 
    - Full Names
    - biography
    - followers count
    - following count
    - is private or not
    - media count
    - external URL
### Run
- In order to run it, clone the repo and run 
```
chmod +x getFollowers.sh
./getFollowers.sh
```
